CHANGED SECTION: Pre-Sprint Preparation (Week 0)
Infrastructure & Tooling Setup (DevOps + Backend):
■ Oracle Cloud account — sign up at cloud.oracle.com (no credit card required for Always Free tier)
 
■ Provision Oracle Cloud VM:
 
 Shape: VM.Standard.A1.Flex
 
 OCPUs: 4, RAM: 24GB, Storage: 200GB boot volume
 
 OS: Ubuntu 22.04 LTS
 
 All permanently free, never expires
 
■ Install on VM: Node.js 20, Nginx, PM2, Git, Certbot (not needed if using Cloudflare SSL)
 
■ Configure Nginx as reverse proxy to port 3000
 
■ Open VM security group ports: 22 (SSH), 80 (HTTP), 443 (HTTPS)
 
■ Neon.tech account — sign up at neon.tech (free, no credit card required)
 
 Create project: "kodetocareer-production"
 
 Copy connection string → set as DATABASE_URL
 
 Enable connection pooling in Neon dashboard
 
■ Upstash account — sign up at upstash.com (free, no credit card required)
 
 Create Redis database: "kodetocareer-cache"
 
 Copy REST URL and Token → set as UPSTASH_REDIS_* env vars
 
 Copy Redis URL → set as UPSTASH_REDIS_URL (for Bull queue)
 
■ Cloudflare account — sign up at cloudflare.com (free)
 
 Add your domain → update nameservers at your registrar
 
 Enable SSL/TLS: Full (strict) mode
 
 Create R2 bucket: "kodetocareer-production"
 
 Create R2 API token with read/write permissions
 
 Add custom domain to R2 bucket: cdn.kodetocareer.com
 
 ■ This replaces: AWS S3 + CloudFront + AWS Certificate Manager
 
■ Resend account — sign up at resend.com (free, no credit card required)
 
 Verify your sending domain (kodetocareer.com)
 
 Create API key → set as RESEND_API_KEY
 
 Free tier: 3,000 emails/month, 100/day

upgrades. Keep account active with monthly login
